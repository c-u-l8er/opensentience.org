OS-002 · Topological Routing · the κ·σ·β thesis

Topology as warrant — structure earns the right to act.

Three structural invariants of an agent's knowledge graph — cycles, conflicts, persistence — don't describe data. They issue warrants. When the graph's shape supplies the cause, a specific operation becomes authorized: not granted by a supervisor, not declared in a policy file, but warranted by where you stand in the structure.

κ proved · 1.9M systems σ via SCOPE · draft β in consolidation routes the Graphonomous loop
I.

Why “warrant,” not “authority”

the precise word

“Authority” is the wrong metaphor for what the substrate actually does. Authority is a standing, role-based power — exactly the thing this mechanism replaces. What the graph issues is narrower, and more honest.

A warrant is issued on cause, authorizes one specific action, and is read off the situation rather than conferred by a superior. That is precisely how the substrate behaves. The system does not hold a standing power to deliberate. It computes, at one point in one neighborhood of the graph, whether the structure supplies the cause — and if it does, the warrant for that single operation issues, here and now. When the cause is absent, no warrant exists, and the operation is simply not authorized.

The word carries a second, useful sense. In epistemology a belief is warranted when it is justified. A topology-warranted operation is a justified one: the justification is structural and checkable, not a confidence score and not an LLM's say-so. Three kinds of structural cause, three warrants:

  • a cycle warrants deliberate — flat reasoning is not authorized where the graph loops back on itself;
  • a conflict warrants reconcile — claims that cannot all hold name exactly what must be resolved;
  • persistence warrants promote — a feature that survived, and is still reinforced, has earned durable memory.
II.

κ — the cycle warrant

authorizes: deliberate

A cycle means the explanation of a node depends, through the graph, on itself. Flat reasoning over such a region systematically misleads — so flat reasoning isn't warranted there.

κ cyclicity warrants deliberate
when κ > 0
κ(G) = min over bipartitions (A,B) of the largest SCC of min(|A→B|, |B→A|) with κ = 0 when the largest strongly connected component has size ≤ 1.

Detection is ordinary graph work: find the largest strongly connected component in the neighborhood (Tarjan), then the minimum bidirectional cut across its bipartitions. The categorical reason it works — why κ > 0 is the same thing as a non-trivial trace in a traced symmetric monoidal category (Joyal–Street–Verity, 1996) — is what tells you the cause is real rather than incidental.

proved  Verified by exhaustive computation across 1,926,351 finite systems (directed graphs n=2–5, finite dynamical systems n=2–7) with zero counterexamples. κ is computed in the Graphonomous route phase today. Full proof: κ — cyclicity.

III.

σ — the conflict warrant

authorizes: reconcile

Cycles tell you the graph has feedback. They don't tell you whether the actors asserting things about a region agree. That needs different machinery — not topology, not category theory, just constraint logic done carefully.

σ conflict warrants reconcile
when σ > 0
σ(G) = |{ (c₁,c₂) ∈ C×C : regions overlap, claim types incompatible }| over the active claims C on regions intersecting G.

Different actors make typed claims on shared regions. An edit over the same span as a reserve is fine; two exclusive claims on the same span are not. Whether a set of claims is jointly satisfiable is the question σ answers — and the load-bearing commitment is that conflicts are derived, never authored, so two implementations compute the same conflict set from the same input. The conflicting pair names exactly which actors disagree, on which region, under which types — which is the agenda reconcile needs.

draft · SCOPE v0.1  σ is the formal name for the deterministic conflict derivation specified by SCOPE, the spatial (&space) primitive. The spec is drafted; the reference implementation is scaffolding. Treat σ as a precise design, not yet a runnable package.

IV.

β — the persistence warrant, with decay

authorizes: promote

A cycle that just appeared differs from one that has held across a hundred consolidations. Treating them alike loses information; treating them alike forever ossifies. The fix is persistence with an explicit half-life.

β persistence warrants promote
when β ≥ θ_β
β(f) = (r(f) − b(f)) · 2^(−(t − r(f)) / τ) birth b, last-reinforcement r, current epoch t, half-life τ.

Persistence accumulates with sustained presence and decays once reinforcement stops, so incumbency can't masquerade as authority — precedent that stops being cited loses force. When β crosses θ_β a feature is warranted to move from fast retrieval memory into durable semantic structure; when it falls below threshold for several consolidations it becomes eligible for pruning.

in consolidation  The Graphonomous consolidate phase already keeps the bookkeeping β needs — birth epoch, last-touched epoch, decay — so this is the lightest delta: name the existing decay signal as an explicit invariant rather than leave it heuristic.

V.

Warrants compose

claimed · not yet measured

Each warrant alone is modest. Taken pairwise and triply they give the substrate a governance vocabulary it otherwise lacks — the legal language is borrowed deliberately.

κ × σ
amendment

A cycle and a conflict on the region. Deliberation is warranted, and the σ-pair tells it exactly which claims to retire and install. The substrate amends itself.

κ × β
precedent

A cycle that has persisted and is still reinforced carries the weight of precedent — similar later queries can cite the prior ruling, unless the landscape shifted. Common law, with decay.

σ × β
durable dissent

A conflict that persists across epochs isn't noise — it's a real feature of the problem. Both claims are preserved as structural facts, not flattened into forced consensus.

κ × σ × β
constitutional evolution

Cycle, conflict, and persistence together: a real, durable, structural disagreement. The strongest signal in the framework — resolution doesn't just seal one decision, it sets a precedent later configurations inherit, until that precedent itself decays. This is the part that needs careful engineering and empirical validation before it claims too much.

VI.

Where the warrant is issued

the route phase

The Graphonomous memory loop has five phases. The warrant is read at route — the phase that decides what the loop is authorized to do next.

retrieve → route → act → learn → consolidate

At route, retrieval has surfaced a neighborhood. The substrate computes κ on that neighborhood; today, when κ > 0, it escalates to deliberation instead of returning a flat answer. As σ and β come online the same phase consults them — a conflict on a spatial sub-graph warrants reconcile, a threshold crossing at consolidate warrants promote. The PULSE manifest can carry the (κ, σ, β) triple as a topology-context token so every loop in the stack reads the same warrants, and PRISM can measure whether warrant-triggered operations actually outperform threshold-triggered ones.

honest status κ issues real warrants in production. σ is a drafted spec. β is decay bookkeeping not yet named as an invariant. The composition vocabulary is a set of claims about what the substrate should do, not yet a measurement of what it has done.
VII.

What's proved, and how to break it

no overclaiming

The framework is shippable in pieces and at different maturities. Naming that plainly is the point.

warrant status basis
κ → deliberate proved Machine-checked over 1,926,351 systems; categorical framing; runs in the route phase.
σ → reconcile draft Deterministic conflict derivation specified in SCOPE v0.1; reference impl scaffolding.
β → promote in consolidation Decay bookkeeping exists in consolidation; not yet a named invariant.
compositions claimed Amendment / precedent / dissent / evolution — argued, not yet measured.

The framework should be testable, so two conjectures worth falsifying first:

  • Cycle-triggering beats uncertainty-triggering. There is a regime where escalating on κ > 0 yields higher post-deliberation accuracy than escalating on output uncertainty, at no higher deliberation cost. If false, cycle presence isn't the right cause and we're back to thresholds.
  • Conflict-aware reconciliation closes conflicts more reliably. When deliberation conditions on the specific σ-pair, it retires the conflicting claims within fewer rounds than deliberation that ignores the structure. If false, SCOPE's conflict record isn't useful as an agenda and σ should only trigger, not inform.
