OS-010 PULSE v0.1

π — Phase Ordering

Proved Property-tested

A compliant cognitive loop moves through its phases in one direction. PULSE's first invariant says that the five canonical phases form a partial order, and no loop may run them backward inside a single cycle. The Invariant Arithmetic chain operation makes this a type-level fact: two values compose only if their phases respect the order.

01The claim

Let phase(v) ∈ {retrieve, route, act, learn, consolidate} carry a total order by index. Composition by chain is phase-graded: it succeeds exactly when the phases do not move backward.

phase order: retrieve ≤ route ≤ act ≤ learn ≤ consolidate L8 (refusal): phase(a) > phase(b) ⟹ chain(a, b) refuses with a π-violation L9 (success): phase(a) ≤ phase(b) ⟹ chain(a, b) succeeds

Conversational traces contain many cycles — each new user message resets the loop to retrieve. The checker segments a trace at those boundaries and requires monotonicity within each cycle, never across them.

02The evidence

π is not proved by exhaustive enumeration (as κ is) but by property-based testing: laws L8 and L9 are asserted against a thousand randomly generated value pairs, and the chain implementation is checked to refuse every backward pair and accept every forward pair.

workbench/app/src/lib/ia/laws.test.ts · L8–L9

fast-check property test, 1000 trials each. L8: any pair with a strictly higher source phase is refused with a phase violation. L9: any forward-or-equal pair composes cleanly.

workbench/app/src/lib/ia/phase_check.ts · checkPhaseOrder()

Runs chain across consecutive edges of a real interaction trace, segmented at user_message boundaries, and reports every backward phase transition. This is the gate that PRISM and the Workbench replay harness use to judge trace integrity.

    Phase ordering is one of seven PULSE v0.1 invariants; covered by the 12-test conformance suite.
  • Canonical phases
    The five phases match the Graphonomous machine architecture exactly: retrieve · route · act · learn · consolidate.

03Verify it yourself

cd workbench/app npm test -- ia/laws.test.ts # L8, L9 — phase-graded chain npm test -- ia/phase_check.test.ts # trace-level monotonicity

04What authority π grants

π grants loop integrity. A loop that can be shown to respect phase order can be trusted to declare its own cadence honestly — which is what lets one loop's output become another loop's input across the [&] · PULSE · PRISM stack without hidden reentrancy. It is the temporal counterpart to κ's structural routing.
